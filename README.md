# Welcome to Bluefinance!

The Bluefinance is a quote monitoring tool developed with Vuejs , Sass and Firebase.


# Installation

## View mode:

    npm start

## Development mode:

    npm install
    npm run serve


## Build

    npm run build


# Data Insert

The data entry is performed by another application.
